About Sprint Training Tool

Overview

Sprint Training Tool is designed for sprinters and coaches to manage sprint training sessions efficiently. It provides precise timing, distance tracking, and start commands to enhance performance.

Main Features

  • Timer: Millisecond-precision timer for each sprint session.
  • Distance Tracking: GPS-based tracking with optional interval voice announcements.
  • Start Commands: "On your mark → Set → Go" with adjustable or randomized delays.
  • Finish Line: Alerts when the target distance is reached.
  • Session Records: Save each session’s time, distance, and notes locally for review.

How to Use

  1. Adjust training parameters: command delays, distance reminders, and target distance.
  2. Click "Start" to begin the session; the tool will announce commands and start the timer.
  3. During training, GPS tracks distance and provides voice updates (if enabled).
  4. Save session records after completion to review progress.

Privacy & Data

Location data is used only to calculate distance. All session records and settings are stored locally on your device. No personal data is shared with third parties.
